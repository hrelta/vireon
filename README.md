# 🌸 Vireon
### Overthink Simulator
*A loop you create in your own mind.*

---

## ✨ About

**Vireon** is a browser-based emotional simulation that turns your thoughts into an escalating overthinking spiral. Type a thought, watch your brain spiral through 5 levels, from rational to completely absurd, then get a sarcastic reality check at the end.

---

## 🎯 Features

- 🧠 **5-Level Thought Escalation** · Rational → Concern → Anxiety → Spiral → Absurd
- 🔍 **Keyword Detection** · Auto-detects social, academic, decision, or default context
- 📊 **Live Progress Bar** · Tracks your spiral in real time
- 💬 **8 Random Final Messages** · Sarcastic but calming reflections
- 🌙 **Dark / Light Mode** · Saved to localStorage
- 📱 **Fully Responsive** · Mobile, Tablet, Desktop

---

## 🛠️ Tech Stack

HTML · CSS · Vanilla JavaScript · localStorage

---

## 🚀 Getting Started

```bash
git clone https://github.com/hrelta/vireon.git
```

Open `index.html` in browser or use **Live Server** in VS Code.
